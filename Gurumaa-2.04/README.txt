This font is created specifically for Linux Operating System. 

A good looking Unicode 6.0 Devanagari compliant font for Linux; created in Archlinux for hindi reading pleasure. 

Make Gurumaa font default for Hindi Language in Linux:

To make Gurumaa font default for Hindi/Devanagari language all over Gnome3, Firefox, Libreoffice and other X11 applications; Copy the fonts.conf to your home folder as .fonts.conf

$ cp fonts.conf $HOME/.fonts.conf
